# Music Playlist Generator

This project generates Spotify playlists based on mood and genre.

## Installation

1. Clone the repo
2. Install dependencies from `requirements.txt`
3. Set up your `.env` file using the `.env.example` as a template.

## Usage

Run the application using: `python app.py`

## Folder Structure

- `app.py`: Main application file.
- `config.py`: Configuration settings.
- `spotify_auth.py`: Spotify authentication logic.
- `playlist_generator.py`: Logic for playlist generation.
- `templates/`: HTML templates for the web application.
- `static/`: Static files including CSS and JS.
- `data/`: JSON files containing mood and genre data.
